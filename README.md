# DropOS

> The operating system for your dropshipping business.

One subscription. Everything you need to source, fulfill, and grow.

---

## Vision

Replace 6–7 fragmented paid tools with a single platform that combines:
- **Analytics** — true net profit, landed cost, multi-store dashboard
- **Fulfillment** — multi-supplier sync, inventory failover, returns portal
- **Sourcing** — product research, supplier scoring, trend finder
- **Marketing** — email flows, review automation, ad tracking

**Target market**: Dropshippers doing $10K–$200K/month, frustrated by $400–$1,000/mo tool stacks.

**Pricing anchor**: One subscription at $99–$149/mo replacing $400–$1,000/mo of tools.

## Stack

- **Frontend**: Next.js (web-first)
- **Backend**: FastAPI + PostgreSQL
- **Auth**: JWT
- **Integrations**: Shopify Partner API, customs tariff APIs (US HTS, EU TARIC)

---

## The Gap We Fill

No tool currently offers at an SMB price point:
1. Real-time multi-supplier inventory failover
2. True net profit tracking (every cost layer: duties, chargebacks, fees, refunds)
3. Multi-store consolidated dashboard
4. Supplier performance scoring + auto-routing
5. Unified returns orchestration
6. Landed cost calculator with live tariff data
